Webseiten

Die Grundidee ist ziemlich einfach:

Das World Wide Web (heute meist kurz: das Web) basiert auf einer Client-Server-Architektur. Der Client ist meist ein grafischer Browser. Er zeigt die Webseiten an, die über das Internet von einem Webserver geladen werden. Zu diesem Zweck wird im Browser eine Adresse eingegeben, die eine bestimmte Ressource im Web anfordert. Das angezeigte Dokument kann Verweise enthalten, die zu weiteren Ressourcen im Web führen.

Fragen

  • Die diversen Elemente html, head und body bilden das Grundgerüst jedes HTML Docs. Das html Element umfasst alle anderen Elemente. Während das Head Element die Metastruktur definiert und den Kopf bildet, welcher im Browserfenstertitel steht. Der Body beinhaltet die eigentliche Datei.
  • Das HTML Element p erzwingt eine neue Zeile. Es spielt dabei keine Rolle, wie der HTML Text dabei aussieht. Sprich, ob er auf einer Zeile ist oder nicht. Das HTML Element h1 erzwingt einen Header. Sprich eine Überschrift. Es existieren bis zu sechs (h6) mögliche Varianten.
  • Ein ASCII Zeichen umfasst 7bit. Ein UTF-8 Zeichen umfasst 8-32bit.
  • Die UTF8 oder Unicode Codierung kann sämtliche Zeichen darstellen, welche im Westlichen Raum existieren. Einschliesslich ü,ä,ö etc.
  • Das src Attribut gibt der Datei an, an welchem Ort in ihrem Ordner, das einzufügende Bild liegt. Das alt Attribut gibt der HTML Datei an, was angezeigt werden soll, wenn keine Bildanzeige möglich ist.
  • Am Besten sind Bilder im TIFF Format geeignet, da der Datenverlust am geringsten ist und das Format mit praktisch allem kompatibel ist.
  • Das <img> Element ist ein leeres Element. Es ist ein leeres Element, wenn kein closing Tag benötigt wird.
  • Ein Tag steht in den Spitzenklammern. Er beinhaltet den Namen eines Elements. Ein Element geht vom Starttag bis zum Endtag.
  • Sie definieren die Art des Elementes. Ob es dick oder schräg geschrieben ist.
  • Es wird durch ein leeres Element, <link>, eingefügt und in die Datei eingebunden mit einem <href> Attribut. Der Type muss mit <type> definiert werden.
  • Mit der Eigenschaft background-color kann die Hintergrundfarbe in CSS definiert werden.
  • Das <title> Attribut bewirkt, dass ein Mouseover mit dem title Text als tooltip angezeigt wird.
  • Das HTML Element <section> definiert Sektionen im Dokument, wie Kapitel, Kopf- oder Fusszeilen. Das HTML Element <article> spezifiziert eigenständigen, autarken Inhalt.
  • Das <header> Tag spezifiziert den Kopf für ein Dokument oder einen Abschnitt. Es ist möglich mehrere header Elemente in einem Dokument einzubauen. Das <head> Element ist ein Gefäss für alle Kopf Elemente eines HTML Dokuments. Es kommt nur einmal vor.
